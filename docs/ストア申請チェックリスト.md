## 1. EAS ビルド前の準備

### 環境変数（EAS Secrets に登録）
```bash
eas secret:create --scope project --name EXPO_PUBLIC_SUPABASE_URL     --value "..."
eas secret:create --scope project --name EXPO_PUBLIC_SUPABASE_ANON_KEY --value "..."
```

- [ ] Firebase Console でプロジェクト作成
- [ ] `google-services.json` を `apps/mobile/` に配置
- [ ] `app.json` の `android.googleServicesFile` パスを確認

### iOS プッシュ通知証明書
- [ ] Apple Developer Console で APNs キー（.p8）を作成
- [ ] Expo Credentials に登録: `eas credentials`

## 3. ストア掲載情報

### 共通
- **アプリ名**: 推し活マーケット
- **サブタイトル**: 推しグッズの売買・交換マーケット
- **カテゴリ**: ショッピング
- **対象年齢**: 17歳以上（App Store） / 17歳以上（Google Play）
- **プライバシーポリシーURL**: https://oshikatsu.jp/privacy
- **利用規約URL**: https://oshikatsu.jp/terms

### アプリ説明文（日本語）
```
推し活マーケットは、ファン同士でグッズを安心・安全に売買・交換できるマーケットプレイスです。

【主な機能】
• AI価格分析で適正価格を自動提案
• エスクロー決済で安心取引
• 信用スコアシステムで誠実なユーザーを評価
• 本人確認（eKYC）で高額取引も安全に
• リアルタイム通知で取引状況をすぐに確認

【対応取引】
• 即時購入・価格交渉
• グッズ同士のトレード（交換）
• プレミアム会員は手数料優遇

古物営業法に準拠した安全な取引プラットフォームです。
```

## 4. App Store 審査対応

| データ種別 | 収集 | 用途 |
|---|---|---|
| メールアドレス | ○ | アカウント作成・認証 |
| 氏名 | ○ | 本人確認（eKYC） |
| 写真/カメラ | ○ | 商品画像アップロード |
| 購入履歴 | ○ | 取引管理 |
| 位置情報 | × | 不使用 |
| 健康情報 | × | 不使用 |

### 審査時の注意点
- [ ] テスト用アカウントを用意して審査メモに記載
  - メール: `reviewer@oshikatsu.jp`
  - パスワード: `ReviewTest2024!`
- [ ] 決済フロー（Stripe）はサンドボックス環境で動作確認済みであることを明記
- [ ] eKYC（TRUSTDOCK）は審査時スキップ可能な旨を記載

## 5. ビルド & 申請コマンド

```bash
# 本番ビルド
cd apps/mobile
eas build --platform all --profile production

# iOS のみ
eas build --platform ios --profile production

# Android のみ
eas build --platform android --profile production

# ストア申請
eas submit --platform ios --latest
eas submit --platform android --latest
```
